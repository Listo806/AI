# AI CRM

This repository contains a minimal Vite + React starter app configured for deployment on Netlify.

Quickstart

1. Install dependencies:

   npm install

2. Run locally:

   npm run dev

3. Build for production:

   npm run build

Netlify

- Build command: npm run build
- Publish directory: dist

Files included

- package.json
- vite.config.js
- index.html
- src/main.jsx
- src/App.jsx
